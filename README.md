## 🚀 Deployment

This site is hosted on **GitHub Pages** (free, unlimited bandwidth, automatic SSL).

To deploy your own copy:
1. Fork this repo
2. Settings → Pages → Source: `main` branch / `/ (root)`
3. (Optional) Add a custom domain via DNS
